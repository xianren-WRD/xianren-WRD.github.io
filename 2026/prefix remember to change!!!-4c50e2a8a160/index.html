AUTOSAR 是什么？

AUTOSAR 全称：

AUTomotive Open System ARchitecture

汽车软件标准架构。

目前主要分：

类型场景
Classic PlatformMCU、ECU、车身控制、底盘、电机
Adaptive Platform智能驾驶、域控制器、高性能计算平台

绝大部分嵌入式岗位：

先学 Classic AUTOSAR

Adaptive 是后续进阶。

学习前置知识

如果这些不会，直接看 AUTOSAR 会崩溃。

C语言

至少掌握：

  • 指针
  • 结构体
  • 函数指针
  • 链表
  • 位操作

推荐：

书籍

  • C Primer Plus
  • The C Programming Language

ARM Cortex-M

理解：

  • 中断
  • NVIC
  • 定时器
  • CAN
  • SPI
  • UART

推荐：

  • STM32

RTOS

重点：

  • Task
  • Queue
  • Semaphore
  • Mutex

推荐：

  • FreeRTOS官网

汽车总线

必须学：

CAN

重点：

  • CAN Frame
  • Arbitration
  • Signal

GitHub 开源项目

因为商业化严重。

1. openAUTOSAR

特点：

  • 开源 Classic AUTOSAR
  • Arctic Core 分支
  • 学习价值最高

2. Arctic Core

Arctic Core v21仓库

最经典开源 AUTOSAR。

很多教学资料都基于它。

3. ARForge
